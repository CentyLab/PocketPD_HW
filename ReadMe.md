# Hardware Overview
- Gerber files - _Gerber.zip; _Gerber.pdf
    - PCB copper and drill file for fabrication
- Schematic files - _Schematic.pdf
    - Board schematic in PDF format
- BOMs - _bom.cvs
    - Bill of material for the PCBA and also the unit assembly
- 3Ds files - 3D Bottom; 3D Knob; 3D Top.stl
    - Model ready for 3D printing for housing and encoder knob
